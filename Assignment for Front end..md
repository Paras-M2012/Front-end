### **Software Engineering Assignment** 

5\) What is software? What is software engineering?

Ans. Software Engineering is the systematic process of designing, developing, testing, and maintaining software using techniques like requirements analysis, design, testing, and maintenance.

&nbsp;	Efficiency: It provides a measure of the resource requirement of a software product efficiently.

&nbsp;	Reliability: It assures that the product will deliver the same results when used in similar working environment.

&nbsp;	Reusability: This attribute makes sure that the module can be used in multiple applications.

&nbsp;	Maintainability: It is the ability of the software to be modified, repaired, or enhanced easily with changing requirements.
